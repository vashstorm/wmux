# New Chat

Start a new AI Assistant chat by clearing the current assistant conversation view and saved voice history.
Use this when the user says things like "new chat", "start over", "clear this chat", or asks to begin a fresh assistant conversation.
This does not affect tmux sessions, windows, panes, projects, config, or AI Logs.
