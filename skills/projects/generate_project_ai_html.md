# Generate Project AI HTML

Generate an AI HTML summary for a project dashboard using the active AI provider.

## Parameters

```json
{
  "type": "object",
  "properties": {
    "project_id": {
      "type": "string",
      "description": "Project ID to analyze."
    }
  },
  "required": ["project_id"]
}
```
