# Sync Project From Tmux

Capture the current tmux session layout into a saved project.

## Parameters

```json
{
  "type": "object",
  "properties": {
    "project_id": {
      "type": "string",
      "description": "Project ID to sync."
    }
  },
  "required": ["project_id"]
}
```
